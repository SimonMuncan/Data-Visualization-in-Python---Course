This is a course on how to visualize data using the Python library Matplotlib.
This course has 4 stages:
1. Stage 1 is how to set up a project and how to install libraries that will be used.
2. Stage 2 is to learn how to manipulate data and get data that we want to plot.
3. Stage 3 is how to plot data using a chart bar.
4. Stage 4 is a final task where you can combine everything that is learned in this course.
